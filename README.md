# Super Smash Search Engine
Ranked and Boolean Search Qury Modes with Disk Index Building
  Built on Maven with VSCode

  Dependencies -> Snowball Tartarus (Port2Stem), slf4j, Spark, Gson, ThymeLeafEngine, jQuery, mapDB.

# Installation Guide
1) download supersmashsearch.jar
2) Run this command in root folder: java -jar supersmashsearch.jar
3) Upon run go to localhost:4567 to test
4) Have a corpus folder in .txt or .json format to upload
5) After hitting submit you will then use the same path to push Query
6) In Search Mode you will enter the search query you want based on the corpus
